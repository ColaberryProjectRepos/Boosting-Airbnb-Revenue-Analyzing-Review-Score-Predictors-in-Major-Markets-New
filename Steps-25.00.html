Creating 4th Chart ( YoY Changes)

Create a measure description table.

·         On the Power BI Desktop Home ribbon, Click on Enter data, name the first column header as Id and the second column header as YoYMeasure and Name the table as YoYTable. In the table, insert ID(1,2,3,4,5) and Measure(ReviewsYoY, CleanYoY, CheckinYoY, CommYoY, LocYoY ) respectively as shown below, then Click ok, Close & apply.  
·         Create a Measure called Variance in YoYTable to display the YoY measures on the Waterfall chart. Use the DAX formula below  
 
Variance = SWITCH(SELECTEDVALUE('YoYTable'[Id]),
1, [.ReviewAvg YOYS],
2, [.CleanAvg YOYS],
3, [.ChkAvg YOYS],
4, [.CommAvg YOYS],
5, [.LocAvg YOYS],
6, [.RevenueAvg YOYS])
