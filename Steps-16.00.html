Adding Project title, Colaberry Logo and AirBnB Logo 16


1.     Insert: Image and select Colaberry logo from the downloaded file
2.     Go to General tab and set Properties: Size: Height = 78, Width = 194
                                                 Position: Horizontal = 0, Vertical = 0
 
 
Adding Airbnb Logo: 

 1.     Insert: Image and Select Airbnb Logo from the downloaded file.
2.     Go to General tab and set Property: Size: Height = 77, Width = 209
                                                 Position: Horizontal = 1070, Vertical = 0
 
 
Adding Title: 
 
 
1.     Insert a Text Box and type the title ‘ Boosting Airbnb Revenue: Analyzing Review Score Predictors in Major Markets’
2.     The Font and style will be set to Segoe (Bold), 18

3.     Go to General tab and set Property: Size: Height = 77, Width = 876
                                                 Position: Horizontal = 194 Vertical = 0
