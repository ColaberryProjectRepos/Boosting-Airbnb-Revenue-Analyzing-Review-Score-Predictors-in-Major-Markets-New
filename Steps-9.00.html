Challenges and Solutions

 

Several Challenges were encountered during the implementation of the project.

1.    Data inconsistency and missing values : This  was addressed through data transformation techniques in Numtra.

2.    Numtra to Fabric Issues : the Jupyter code generated in Numtra was not compatible with Microsoft Fabric. Thus the Jupyter code generated in Numtra had to be converted to Spark/python which is compatible with Microsoft Fabric. 

3.    DAX YoY calculation : This was another challenge to get the correct Year over Year results for the sliced and diced data.

4.    Images in Slicer : Insert images in Slicer was yet another challenge. After some study and research I was able to figure out the different ways that involved several steps like  finding the images, saving it, creating links to those images. Creating a table and then populating these image links in it. Creating relationship with other existing table. Make certain settings and finally access the images in the New Slicer provided in Power BI.

 

 

 

  

 Lessons Learned

 

Continuous model refinement and integration of real-time feedback are necessary for sustained accuracy and relevance. Effective communication with stakeholders is essential for turning data into actionable insights and making informed decisions. Presenting data as a narrative helps stakeholders understand its context and relevance. Clearly communicating the business benefits ensures stakeholders see the value and impact of the insights. Tailoring the communication to the audience's needs makes the insights accessible and relevant. Building trust through accurate data and using clear visualizations simplifies complex information, enabling confident decision-making.
