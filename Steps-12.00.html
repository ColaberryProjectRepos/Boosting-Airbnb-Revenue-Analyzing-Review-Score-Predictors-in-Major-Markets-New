Creating KPIs and sub KPIs using DAX Measures 12
Creating DAX Measures

1.     Click on Modified_AirBnB_Final table and select New Measure
2.     Start writing the DAX formula (all the formulas are at the bottom) or copy and paste one by one.
3.     After you done click on the green right mark so it will be saved.
4.     Repeat the steps until all the measures are created.
 

1.   Total Review = COUNT(Modified_AirBnB_Final[prediction])
 
2.   Reviews Per Month = DIVIDE([.Total Review], DISTINCTCOUNT('DimDateTable'[Month Name Short]),0)+0
 
3.   ReviewAvg YOYS = 
VAR SelectedYears = VALUES('DimDateTable'[Date].[Year])
RETURN
       AVERAGEX (
        SelectedYears,
 VAR CYClean = CALCULATE ( AVERAGE(Modified_AirBnB_Final[prediction]), 'DimDateTable'[Date].[Year] = EARLIER ( 'DimDateTable'[Date].[Year] ) )
VAR PYClean = CALCULATE ( AVERAGE(Modified_AirBnB_Final[prediction]), 'DimDateTable'[Date].[Year] = EARLIER ( 'DimDateTable'[Date].[Year] ) - 1 )
        RETURN
            IF (
                ISBLANK ( PYClean ),
                0,
                DIVIDE ( CYClean - PYClean, PYClean )
            )
       )
